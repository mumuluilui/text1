 <h3>招式</h3>
<ol>
  <li>龟派气功 ――继承至：龟仙人 效果：双手相靠，凝聚气于手心，用自己的气痛击敌人</li>
  <li>超。龟派气功――继承至：自己原创 效果：大致上和龟派气功相同，就是力量增加了许多。</li>
  <li>界王拳――龟派气功――继承至：自己原创 效果：界王拳是将能力加倍的神功，这一招也可以将龟派气功的能力加倍，可惜悟空只用过一次，而且只到4倍界王拳。
</li>
  <li>十倍龟派气功――继承至：自己原创 效果：在能力加强后已经能提升，并且改良龟派气功，这个就是改良版本，在龙珠GT里面，超4状态的悟空经常使用。
</li>
  <li>超大型龟派气功――继承至：自己原创 (龙珠GT里面，超4状态和贝吉塔融合之后) 效果：两个字，暴强！</li>
  <li>界王拳―― 继承至：界王 效果：将自己的力量加倍提升，比如说用界王拳二，力量就提升两倍。
</li>
  <li>二十倍界王拳―― 继承至： 自己原创 效果：不用说你们应该清楚了，将自己的力量翻二十倍。
</li>
  <li>元气弹(大，中，小)―― 继承至：界王 效果：集合元气，悟空最强的攻击，有摧毁一切的威力。
</li>
  <li>太阳拳―― 继承至：天津饭(看了一遍就学会了) 效果： 绽放有如太阳一般的光，让对手暂时失去视力。
</li>
  <li>残像拳―― 继承至：无(无缘无故就会了) 效果： 制造出许多分身，但是却都能运动自如，不过没有破坏力，只是为了混侥对手的视听。
</li>
  <li>龙拳爆发―― 继承至：无(原因和残像拳一样) 效果：集合了所有能源力量的一击，志在一击杀敌。</li>
</ol>


  <h2>超级赛亚人之四</h2>
  <p>《龙珠GT》（非鸟山明原创但是得到许可的动画作品）中孙悟空再度变强原创的变身，金发变为黑色披肩发，黑色眼圈，身体没有火焰，覆盖红色体毛，身体更加强壮。</p>
  <p>生平经历出世后被遗弃在深山﹐被孙悟饭发现后带回去当孙子养。天生性格纯品﹑正直﹐而心境也很清纯﹐也曾因为这样而得救却也多次闯祸和吃亏。胃口特大﹐曾经吃过五十人份的菜。变成大猩猩会完全失去理智，在他小时候的一次变身中意外地踩死爷爷孙悟饭﹐因此之后一直独居山中。十分崇拜悟饭留下的四星球﹐所以会为了它而展开搜集龙珠之旅。弱点就在尾巴﹐只要被人捉住那里便会全身无力﹐不过断了几次尾后这个弱点也在严格的修鍊之下被克服了。十二岁时遇见搜集龙珠的布尔玛并离开深山去闯荡外面的世界﹐展开了难忘的大冒险。后来投入龟仙派跟龟仙人学武八个月﹐更在第廿一届天下第一武术大会中的总决赛跟程龙比武﹐结果得到亚军。再次出发找龙珠时因惹上了红带军而不断锻鍊﹐更在同伴的帮忙之下把白队﹑蓝队和黄队一一消灭。被桃白白打败后便爬上了凯琳塔﹐并在猫仙人的教导下功力大增﹐最后成功地击败桃白白再将红带军消灭。后来为了令乌巴的爸爸复活而随同同伴往占星婆婆处决战﹐更跟爷爷孙悟饭再度见面。随后的修行中粉碎了天龙的野心并且从天津饭手中救出了猪鹿蝶。在第廿二届天下武术第一大会中跟天津饭争冠军时再次得到亚军﹐之后便为了被魔族杀害的无限等人而跟笛子大魔王苦战。惨败其手后便为了增强而饮了超神水﹐并在克服毒素后功力大增﹐于是便在一番激战后成功地打倒笛子大魔王﹐而且还上了天界来令龙珠复活。当一切平息后便为了三年后和大魔王二世——魔童的决斗而留在天界跟天神修鍊﹐后来也曾经走到魔界和穿越时空。在第廿三届天下第一武术大会的总决赛中跟魔童决战。打了一场惊天动地的比武后终於将魔童打倒。后来便跟小时候的未婚妻琪琪结婚。 

</p>
  <p>龙珠完结五年后从哥哥拿迪斯口中知道自己是几乎灭族的战斗民族撒亚人卡卡罗特﹐而且是为了消灭地球人才在出生不久被人从故乡比达星送到地球﹐不过因小时候跌伤头部而忘记一切过去的记忆和指令。为了救回被拿迪斯捉去当人质的儿子孙悟饭﹐只好跟宿敌笛子魔童联手出击。最后成功地杀死拿迪斯﹐但自己也牺牲了。因为一年后两个更强的撒亚人会攻击地球﹐所以死后便去找北银河的界王修鍊﹐更为了去界王星而在阴间的蛇道上走了大半年。修鍊完后以龙珠复活并赶回地球。在之后的大战中虽然被比达弄得体无全肤﹑满身伤痕﹐但在大家的协助下总算解救了地球。一个月后终於痊愈﹐更坐小时候来地球的太空船到被杀的魔童故乡娜美星支援到那里找龙珠的悟饭们﹐并且在飞行的六天内大幅提升战斗力。在娜美星上对抗菲利的手下杰克时被交换了身体﹐要靠比达帮手才能还原及疗伤。跟菲利大战时完全落下风﹐连最后希望元气弹也失败。不过后来因无限的死而变成传说中等超级撒亚人。在跟著的战斗中击败了菲利和坐太空船逃出快爆炸的娜美星﹐然后到也特他星住了一年来学他们的特技。回地球后得到未来杜拉格斯的药和提点﹐所以便为了对付三年后会出现的人造人而跟悟饭和魔童不断修。在人造人的追捕和斯路的暴行中因那心脏病病发而用了那药来避过心脏病死的命运﹐之后休息了数天。醒来后请悟饭﹑比达和杜拉格斯上天界的精神时光屋来突破超级撒亚人的界限。斯路游戏中斯路进行激烈的决战。之后为了救地球﹐于是将快自爆的斯路带上界王星﹐结果被炸得粉碎。不过随后在阴间传精神力量给悟饭﹐让他成功地消灭斯路。可惜因为已复活这而不能复活﹐所以之后的七年内在阴间修鍊。七年后在天下第一武术大会当天暂时还阳﹐但是中场却要跟界王神一起对付巴菲迪和魔人布欧﹐结果和比达打了一场大战。为了拖延布欧而使用超级撒亚人３变身﹐但却导至还阳时间大大缩短﹐所以在教了悟天和杜拉格斯融合后回到阴间。在跟著的一段时间留在界王神界陪正在修鍊的悟饭。当和布欧战斗的悟饭出现危机时得到老界王神的生命而复活﹐于是马上回到战场。在地球上跟比达合体成比古洛﹐将布欧打得无力反抗。两人在布欧体内分开后救了被布欧吸收了的人出来。地球毁灭后到界王神界战原本的布欧﹐最后集齐大家的元气将他彻底消灭。跟著的十年中在地球过平静的生活。在另一个未来中在跟人造人战斗前因也特他星的病毒性心脏病而死。 

</p>
  <p>自从带欧布离开天下第一武术大会后便和他到处修鍊了五年。当和欧布在天界打完最后的一战后便被毕拉夫意外地用最强的黑星龙珠变回小孩的模样。之后为了找四散银河的黑星龙珠来拯救地球而跟阿包和杜拉格斯坐太空船上宇宙。虽然还是一个大人﹐但没有黑星龙珠的力量的话便不能变回原来的状态。由于返老还童﹐所以性格变得有点孩子气﹐而且力量也下降了。在宇宙里第一个降落的星球依米加惑星上跟这次冒险第一个强敌尼哲决战﹐更在变小后首次使出超级撒亚人能量克敌。在幪玛斯惑星和加鲁多星上找到龙珠后在跟踪巴啦巴拿兄弟的途中和比巴星的姆玛大战﹐并且找到克服它们的方法。去鲁顿星球时负责看守巴啦巴拿兄弟﹐而且还消灭了莫斯和尼奥。跟木斯决战了一会后便跟杜拉格斯联手战顿鲁。虽然开始时占了优势﹐但鲁顿变成最强的状态后陷入劣势。幸好在最后关头跟鲁顿体内的阿包用特别的攻略一起将他毁灭。在M2星球被米加粒子炮部队捉去后因为阿包被超级米加粒子阵打倒而大发雷霆﹐后来驶出真正的功力消灭他们。在跟列顿将军的大决战中勉强打平手﹐却因为阿包的出现而分心﹐结果被变成金属板。被机露放后打倒了缪博士和Baby﹐后来在遇难船和必达星等其他星球战复活的Baby。回地球后不久便被家人和比达Baby袭击﹐随后以惨败收场﹐幸好在中歼悔死亡弹前被杰比度界王神救起。不幸掉到蜀咕罗空间受了不少苦头后平安地回界王神界﹐再在界王神们的一番帮助下长回尾巴。到达舒夫智星后因看见地球而变成一只发狂的大猩猩﹐不过看见流泪的阿包后便变成最强的超级撒亚人4。跟大猩猩Baby苦战后完全落下风﹐但吸收了其余四个超级撒亚人的能量后也能够把Baby从比达身体赶走并消灭。多年后接受了缪博士和基洛博士的挑战书﹐去了地狱找他们﹐但却以被封印地狱收场。费尽心机封印打不死身的斯路和菲利后得到魔童的帮助才能回地球﹐并且马上决战超17号。拳力和十倍龟波气功全部被他吸收后在危急关头中得到18号的帮助﹐于是找到打倒超17号的方法。邪恶龙在地球散播后便再次开始寻龙珠之旅。把二星龙﹑五星龙和六星龙收伏后把阿包从七星龙救出再把他消灭。之后跟四星龙进行激烈的决战﹐但在中途被三星龙阻挡。虽然最终也收伏了三星龙﹐但双目也被他抓盲﹐结果被一星龙重重地压倒﹐但吸收了年青的超级赛亚人能量后便打倒了一星龙。不过后来被超一星龙打败﹐幸好回复视力后比达及时出现﹐于是两人便融合成格比达。当两人分开后为了保护四星珠而将它吞下﹐然后再跟四星龙联手对付一星龙。当一星龙再变回最强状态后便在大家的帮忙下集合有史以来最大的元气弹将他消灭。跟神龙离去后到处探访旧朋友﹐后来变成了龙珠一分子。一百年后以幻影来见子孙悟空Jr﹐然后在武术大会的日子回来。   

</p>

    <dd>一个坚强不屈，勇敢无匹，十分友善，有着一颗仁慈的心，并且掌握地球命运的人，这就是孙悟空。悟空往往是那么的善良，但是却又是出乎人们意料之外的笑料，由于长年居住与深山当中，就连什么是[女孩]都不知道，也没有见过，只是他爷爷说过：“对女孩要温柔。”他有点孩子气，还带着傻气，即使成长为一个强者，这个性格还是没有改变，他十分优哉，不管面对什么强敌他都不怕，也不退缩，反而露出无比兴奋的神色，这个表情仿佛在无言之中告诉我们“我就是孙悟空！放马过来！”由于小时候撞到了脑部，让他从一个凶恶的孩子变成了一个乖巧听话，容易相处，没有邪念的人，但是同时也让他变得呆头呆脑，还有点迟钝。哈，真是可喜又可忧啊。</dd>
